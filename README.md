# Sistema Inteligente de Orientação Financeira para Iniciantes

Aplicação web desenvolvida para auxiliar investidores iniciantes a compreender seu perfil de risco, receber recomendações de carteira, simular cenários de investimento e acessar conteúdos educativos sobre finanças.

## Link do sistema

https://orientacao-financeira.streamlit.app/

## Objetivo

O objetivo do projeto é reduzir a complexidade do processo de entrada no mercado financeiro, oferecendo uma experiência guiada, visual e educativa para usuários iniciantes.

## Funcionalidades

- Questionário de perfil do investidor
- Classificação em perfil conservador, moderado ou agressivo
- Recomendações de investimentos por perfil
- Montagem personalizada da carteira
- Simulação com cenários pessimista, base e otimista
- Gráficos e tabelas de resultados
- Explicações sobre juros compostos e controle emocional
- Área educacional com recomendações e conceitos básicos

## Tecnologias utilizadas

- Python
- Streamlit
- HTML
- CSS
- JavaScript
- GitHub
- Streamlit Community Cloud

## Funcionamento do sistema

O usuário inicia respondendo a um questionário sobre risco, experiência e horizonte de investimento. Com base nas respostas, o sistema classifica seu perfil e apresenta uma carteira recomendada.

Em seguida, o usuário pode ajustar os percentuais de cada ativo e simular a evolução do patrimônio ao longo do tempo. A simulação considera aportes, juros compostos e três cenários de retorno.
